# Oriel Graph

A graphing calculator with useful calculation tools

## Features

 - Calculate zeros and local min/max of a function
 - Table of values
 - Save graph as image
 - Radian scale
 - Customizable line colour, style and weight
 - Graphs sin, cos, tan, log, ln and sqrt functions
